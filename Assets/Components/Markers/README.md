# Markers

This component allows for visualization of various [markers](https://wiki.ros.org/rviz/DisplayTypes/Marker) in the system.

## Markers

| Type | Description | Visualization |
|------|-------------|----------------|
| `Arrow` | A simple arrow that can be used to visualize a direction. | ![Arrow](/docs/images/marker_arrow.png) |
| `Cube` | A simple cube that can be used to visualize a position. | ![Cube](/docs/images/marker_cube.png) |
| `Sphere` | A simple sphere that can be used to visualize a position. | ![Sphere](/docs/images/marker_sphere.png) |
| `Cylinder` | A simple cylinder that can be used to visualize a position. | ![Cylinder](/docs/images/marker_cylinder.png) |
| `Line Strip` | A line that connects various points. | ![Line Strip](/docs/images/marker_line_strip.png) |
| `Line List` | A collection of line segments. | ![Line List](/docs/images/marker_line_list.png) |
| `Cube List` | A collection of cubes. | ![Cube List](/docs/images/marker_cube_list.png) |
| `Sphere List` | A collection of spheres. | ![Sphere List](/docs/images/marker_sphere_list.png) |
| `Points` | A collection of points. (NOTE: its more performant to use the PointCloud2 Lidar interface for now) | ![Points](/docs/images/marker_points.png) |
| `Text` | A simple text label that faces the user | ![Text](/docs/images/marker_text.png) |
| `Mesh` | Not currently supported... |  |
